# A Image conversion tool in the Terminal
## Usage
```
usage: 
convert.py [-h] [-t {png,jpg,jpeg}] [-o OUTPUT] filename [filename ...]

positional arguments:
  filename

options:
  -h, --help            show this help message and exit
  -t {png,jpg,jpeg}, --type {png,jpg,jpeg}
                        Type to convert to
  -o OUTPUT, --output OUTPUT
                        Outputs to a specified folder
```
